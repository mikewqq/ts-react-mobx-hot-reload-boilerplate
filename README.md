# ts-react-mobx-hot-reload-boilerplate

boilerplate with
 - react
 - typescript
 - webpack
 - hot reload
 - mobx-state-tree
 - support production build caching and vendor code splitting
 
 
## Usage
install dependencies
```
npm i
```

start dev server with hot reload
```
npm run satrt
```


make production build
```
npm build
```
